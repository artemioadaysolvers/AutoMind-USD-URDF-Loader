AutoMind URDF+ Viewer BUILD139

Merged from URDFPlus_Viewer_Reparado.html / BUILD138 into the modular URDF_Viewer package.

Main fixes:
- parseVec now supports fallback=null for optional URDF+ vectors such as axis_world / axis_joint.
- Collada/texture URL resolution now blocks invalid file:// and data:model/*.jpg requests and returns a transparent fallback texture instead of breaking the load.
- package://, meshes/, texture(s)/ and basename asset matching are preserved for zipped robots such as quadruped/ANYmal-style packages.
- URDF_Zip is always merged with explicit assetDB entries, instead of being ignored whenever assetDB is non-empty.
- URDF text lookup now decodes text from base64/data URLs when necessary.
- Keeps the existing modular viewer features: ComponentsPanel, ToolsDock / Viewer Tools, section clipping, render modes, isolation/show-all, thumbnails, hover/selection and constrained drag plumbing.
